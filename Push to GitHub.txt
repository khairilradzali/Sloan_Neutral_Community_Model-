1️⃣ Initialize Git in your folder

Open PowerShell in your repo folder:

cd "C:\Users\User\OneDrive\Desktop\RandomForest_Microbiome"
git init

2️⃣ Add remote repository

Go to GitHub → New repository → name it RandomForest_Microbiome.

Copy the HTTPS URL, e.g., https://github.com/yourusername/RandomForest_Microbiome.git.

Link your local repo:

git remote add origin https://github.com/yourusername/RandomForest_Microbiome.git

3️⃣ Stage and commit files
git add .
git commit -m "Initial commit: Random Forest 16S analysis scripts and README"

4️⃣ Push to GitHub

If your default branch is main:

git branch -M main
git push -u origin main


If you get a credentials prompt, enter your GitHub username and PAT (personal access token) instead of a password.
